Kafka:

docker pull ericdock/docker-kafka
docker run --name listenbrainz-kafka -d -p 2181:2181 -p 9092:9092 ericdock/docker-kafka

Postgres:

docker pull postgres
docker run --name listenbrainz-postgres -e POSTGRES_PASSWORD=listenbrainz -d postgres

Redis:

docker pull redis
docker run --name listenbrainz-redis -d redis

Listenbrainz

docker build -t listenbrainz:latest .
docker run --name listenbrainz -p 8080:8080 -d listenbrainz
