API Documentation

If you would like to use our API to submit/fetch listens from ListenBrainz, please take a look at API documentation.

Client Libraries

Client libraries have already been written for some languages:
  • Haskell: listenbrainz-client
  • Go: go-listenbrainz

Listen Submitters

These are programs that submit listens to ListenBrainz.
  • mpd: libra
  • Lollypop, a modern music player for GNOME
  • Last.fm Scrobbler, an extension for Firefox and Chrome
  • Simple Last.fm Scrobbler, for Android devices
