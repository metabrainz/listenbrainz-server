Live Data

As an alternative to a huge download which can be hard to manage, you may also query the ListenBrainz data live on Google's BigQuery service. We stream data to this dataset in real time and every user gets a free query allowance from Google, so this is by far the easiest way to explore the ListenBrainz data!

Explore ListenBrainz data on BigQuery.
